# PositionLab WebApp
---
This is an a tool for trades to manage their operations and make a journal of all his operations then they can see back what they did wrong and learn from that, it also make calculations fast for you of how much to put in each trade and when to doo the trade. It still developing but is usable already.

---

# Tech Stack

- React
- Typescript
- Vite

---

# Features

- You can save your operations on a historial
- It makes caclulations for you to improve the workflow
